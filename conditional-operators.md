
# 
| Operator | Meaning                  |
| -------- | ------------------------ |
| `==`     | Equal to                 |
| `!=`     | Not equal to             |
| `>`      | Greater than             |
| `<`      | Less than                |
| `>=`     | Greater than or equal to |
| `<=`     | Less than or equal to    |

| Operator    | Meaning       |
| ----------- | ------------- |
| `X` `&` `Y` | `X` *and* `Y` |
| `X` `|` `Y` | `X` *or* `Y`  |
